# Petal Pushers

## Setup

### Backend

1. Install Python (I use v3.12.7)
2. `cd backend`
3. `python -m venv venv`
4. `venv\Scripts\activate`
5. Install dependencies with `pip install -r requirements.txt`
6. Run the app.py with `python app.py`, or through the VSCode extension

### Frontend

1. Install node.js
2. `cd frontend`
3. `npm install`
4. `npm run dev`
5. Access the page through <http://localhost:5173/>
